# Labs - Índice General

**Estructura**: Organizados por nivel  
**Propósito**: Práctica hands-on de cada tema  
**Entrega**: Código + evidencias + cleanup

---

## Estructura

```
labs/
 lab-templates/         # Plantillas para crear labs
 basico/               # 6 labs (Semanas 1-2)
 intermedio/           # 9 labs (Semanas 3-4)
 avanzado/             # 8 labs (Semanas 5-6)
```

---

## Cada Lab Incluye

```
lab-nombre/
 README.md                     # Instrucciones
 evidencias-checklist.md      # Qué capturar
 cleanup.sh                    # Limpieza
 assets/
    template.json
    policy.json
    [otros archivos necesarios]
```

---

## Checklist por Lab

Antes de marcar como "done":

- [ ] Instrucciones claras
- [ ] Costo estimado
- [ ] Cleanup tested
- [ ] Screenshots provided
- [ ] Min 3 evidencias
