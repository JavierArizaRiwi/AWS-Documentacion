## Escala de Puntuación

```
90-100 = Star-5 Excelente (Publication Ready)
80-89  = Star-4 Muy Bueno (Production Ready)
70-79  = Star-3 Bueno (Completado)
60-69  = Star-2 Satisfactorio (Funcional)
50-59  = Star-1 Necesita Mejora (En progreso)
< 50   = [X] No completado
```

**Nota Final = Promedio ponderado**

---

## Cómo Usar Rúbricas

1. Revisor abre rúbrica correspondiente
2. Lee criterios del 1 al N
3. Asigna puntuación (Novato/Intermedio/Avanzado)
4. Suma total
5. Proporciona feedback constructivo
